🔥 Firebase Setup (Free — takes 3 minutes)

1Go to console.firebase.google.com → Add project → name it (e.g. badminator) → Continue → disable Analytics → Create
2In your project → click Realtime Database (left sidebar) → Create Database → choose any location → select "Start in test mode" → Enable
3Your database URL appears at the top: https://your-project-default-rtdb.firebaseio.com — copy it
4Paste the URL below, set your admin password, click Save & Test
5Host the HTML file on Netlify Drop → share the URL with everyone. Scores sync instantly for all viewers! 🚀
